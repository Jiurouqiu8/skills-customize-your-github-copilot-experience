# 📘 Assignment: Python Unit Testing with pytest

## 🎯 Objective

Learn how to write automated tests in Python using pytest. Students will practice testing functions, handling assertions, and verifying both normal and edge-case behavior.

## 📝 Tasks

### 🛠️ Write tests for helper functions

#### Description
Use pytest to write tests for the provided helper functions.

#### Requirements
Completed assignment should:

- Create a `test_helpers.py` file with pytest test functions
- Test at least two helper functions from `starter-code.py`
- Verify both expected outputs and at least one error or edge case

### 🛠️ Add tests for input validation

#### Description
Write tests that confirm the helper functions handle invalid or unexpected input correctly.

#### Requirements
Completed assignment should:

- Include tests that pass invalid values to helper functions
- Use `pytest.raises` for functions that should raise exceptions
- Confirm the error message or exception type is correct

### 🛠️ Run pytest and review results

#### Description
Run the test suite using pytest and fix any failing tests.

#### Requirements
Completed assignment should:

- Run `pytest` successfully in the assignment directory
- Show output for at least one passed test and one failed test before fixing it, if applicable
- Optionally add comments explaining what each test covers
